## ✨ 主な機能

**ニュース収集**
- 15メディアのRSSを同時取得・マージ
- 新しい順（降順）で自動ソート
- タイトル正規化による重複除去（記号・表記ゆれにも対応）
- 10分間キャッシュで無駄なリクエストを削減

**センチメント判定**
- 各記事を「▲ 強気 / ▼ 弱気 / → 中立」にキーワード自動判定
- タブに強気・弱気の件数バッジを表示

**自動更新**
- 15分ごとにバックグラウンドで自動再取得
- 28秒ごとに未表示記事をフィード先頭に追加（タイムライン的な流れ）
- 次回更新まで残り時間をカウントダウン表示

**手動更新**
- タブごとの「↺ 更新」ボタン：そのセクターだけ即時更新
- 「↺ 全セクター更新」ボタン：全9セクターを一括更新

**その他**
- ニュースカードをクリックすると元記事が新しいタブで開く
- BTC/USD価格をリアルタイム表示（CoinDesk無料API）

## 🔧 仕組み

ブラウザから直接RSSを取得しようとすると**CORSエラー**が発生します。
このアプリは**無料のCORSプロキシ**を経由することで解決しています。

```
ブラウザ → CORSプロキシ → 各メディアのRSSフィード
```

プロキシは3つ用意しており、1つが応答しない場合は次に自動切り替えします。

| プロキシ | URL |
|--------|-----|
| 第1候補 | api.allorigins.win |
| 第2候補 | corsproxy.io |
| 第3候補 | api.codetabs.com |

## ⚙️ カスタマイズ

HTMLファイルをテキストエディタで開くと、以下の箇所を自由に変更できます。

### ニュースソースを追加・変更する

```javascript
const RSS_FEEDS = [
  { id:'nikkei', label:'日経', url:'https://www.nikkei.com/rss/all.rdf' },
  // ↑ id（内部ID）, label（表示名）, url（RSS URL）を追加するだけ
];
```

### セクターのキーワードを変更する

```javascript
{ id:'ai', name:'AI・半導体',
  kw:['AI','半導体','NVIDIA', ...] }  // ← ここに追加・削除
```

### 自動更新の間隔を変える

```javascript
const AUTO_REFRESH_MS = 15 * 60 * 1000;  // 15分（ミリ秒で指定）
const INJECT_MS       = 28 * 1000;        // 28秒ごとにカード追加
```

## ❓ よくある質問

**Q: ニュースが表示されない / 少ない**
→ CORSプロキシが一時的に不安定な場合があります。「↺ 更新」ボタンを押すか、しばらくしてから再試行してください。

**Q: 同じ記事が重複して表示される**
→ まれに異なるRSSから同じ内容が配信される場合があります。「↺ 更新」で再取得すると解消します。

**Q: スマホで表示が崩れる**
→ レスポンシブ対応済みですが、一部の古いブラウザでは表示が異なる場合があります。Chrome/Safariの最新版を推奨します。

**Q: 記事が古い / リアルタイムでない**
→ RSSの配信タイミングは各メディアに依存します。速報性が必要な場合は、時事通信・共同通信・NHKのタブを優先してご確認ください。

**Q: GitHub Pagesで公開後、スマホからアクセスするには？**
→ `https://あなたのGitHubID.github.io/リポジトリ名/` にスマホでアクセスしてください。SafariなどのブラウザでURL欄下の「共有」→「ホーム画面に追加」でアプリのように使えます。
