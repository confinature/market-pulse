# 📈 MARKET//PULSE
クロードちゃんに作ってもらった記念。

https://confinature.github.io/market-pulse/

日本株の9セクターに関するニュースを、国内15メディアのRSSから自動収集してタイムライン表示する。
（見た目がカッコいい、単なるRSSリーダー）
 
- 記事の本文は取得・保存していません（タイトル・概要・URLのみ表示）
- 記事をクリックすると必ず元のメディアサイトへ誘導します
- データはブラウザ内にのみ一時的に保持し、外部サーバーへの送信は行いません
- 各メディアのRSSサーバーへのリクエストはユーザー操作・15分間隔に限定しています
- 非商用・個人利用のプロジェクトです
- 記事の著作権は各メディアに帰属します
- 収益化を目的としていません
